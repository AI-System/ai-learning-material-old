Python 中的数据类型和运算符
---

### 概述

- 数据类型：整型、浮点型、布尔型、字符串、列表、元组、集合、字典

- 运算符：算术、赋值、比较、逻辑、成员、恒等运算符

- 内置函数、符合数据结构、类型转换

- 空格和样式指南

### 算术运算符

- + 加
- - 减
- * 乘
- / 除
- % 取余（相除后的余数）
- ** 取幂（注意 ^ 并不执行该运算，你可能在其他语言中见过这种情形）
- // 相除后向下取整到最接近的整数
